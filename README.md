## Configuration
Create "config.json":

```
{
    "apiKey": "YOUR_API_KEY_HERE"
}
```

To get your API key, follow instructions on [Trafikverket](https://www.trafikverket.se/e-tjanster/trafikverkets-oppna-api-for-trafikinformation/).
